View-Dependent Rendering in Unity

View-Dependent Rendering in Unity was the dissertation I worked on during university, where I created a low poly game that used a webcam to track the player's position and control a spaceship in real time. The player's movement in front of the webcam determined the position of the spaceship on screen, adding a physical and interactive element to the gameplay. The objective of the game was to avoid incoming asteroids and safely navigate back to the home planet. I built the game using Unity and C#, combining camera input with game mechanics to create an engaging and responsive experience.

Skills: Unity, C# , Python, OpenCV in Unity, Blender, Agile, Visual Studio, Code Testing, Unit / Integration / Performance Testing, User Feedback.

DreamNest

DreamNest was a project I participated in during university it was a group project in which we developed a ecommerce website that sold room enhancement items such as artworks, lamps and other room decor items to customers who want to elevate the look of their rooms.
